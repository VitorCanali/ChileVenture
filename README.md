# 🌎 ChileVenture — Landing Page (Prova Olímpica)

## 📌 Sobre o projeto

Este projeto foi desenvolvido como parte de uma **prova no estilo olímpico**, com o objetivo de avaliar habilidades em desenvolvimento web (HTML e CSS).

Para mim, além da avaliação, esta atividade também está servindo como uma **revisão prática dos conceitos fundamentais de front-end**, especialmente estruturação de layout e estilização.

---

## 🚧 Status do projeto

⚠️ **Parcialmente concluído**

Até o momento, foi desenvolvida **apenas a primeira parte do site**, ou seja:

* A **primeira tela (landing/hero section)**
* Estrutura básica de navegação (navbar)
* Estilização inicial com CSS
* Uso de imagem de fundo com overlay
* Organização centralizada de conteúdo

---

## 🖥️ Preview da funcionalidade implementada

A primeira tela apresenta:

* Logo e menu de navegação
* Botão de destaque ("COMPRAR")
* Título principal chamativo
* Texto descritivo
* Botão de ação ("VER MÁS")
* Background com imagem + efeito de escurecimento (overlay)

---

## 🛠️ Tecnologias utilizadas

* HTML5
